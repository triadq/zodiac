2. THE CALCULATION (MASKING)
To find the symbol, add your Word Length + Vector Digit. Take the last digit of the sum.
Example: Word "Blue" (4) + Digit (9) = 13. Last digit is 3. Symbol is #.

3. THE SHUFFLE (EVEN vs ODD)
EVEN Digits (0, 2, 4...): Keep it natural.
Head(Cap) + Tail(Low) + Digit + Symbol

ODD Digits (1, 3, 5...): Flip it inside-out.
Digit + Symbol + Head(Cap) + Tail(Low)

4. THE PRISM (SERVICE DERIVATION)
To create a password for "Dropbox" (7 letters) without this tool:
1. Count the letters in the service name (7).
2. Find the 7th Segment of your Master Key.
3. Append the Service Name (Capitalized).
Result: [Segment 7] + Dropbox
