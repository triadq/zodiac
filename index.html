SURVIVAL MANUAL (BLACK HOLE PROTOCOL)
1. BRAID: Pair every Digit of your Vector with a Word from your Text.
   (If you have more digits than words, loop back to the first word. They never collide.)
2. MANTRA: For each pair, write 4 chars: HEAD(Cap) + TAIL(Low) + DIGIT + SYMBOL(Len).
   Ex: "Fox" (Len 3) + Digit 5 -> F + x + 5 + # -> Fx5#
3. MAP: 1=! 2=@ 3=# 4=$ 5=% 6=^ 7=& 8=* 9=( 0=)

GRID RECOVERY (DIAGONAL SPREAD):
- Segment 1: Row [Digit 1], Column 0
- Segment 2: Row [Digit 2], Column 1
- ... Segment N: Row [Digit N], Column [N-1 % 10].
