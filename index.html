SURVIVAL MANUAL (BLACK HOLE PROTOCOL)
If this paper is lost, reconstruct the password in the dirt.

1. THE BRAID: Pair every Digit of your Vector with a Word from your Text.
2. THE MANTRA: For each pair, write 4 characters: Head + Tail + Digit + Symbol.
   - Head: First letter (Uppercase).
   - Tail: Last letter (Lowercase).
   - Digit: The Vector digit.
   - Symbol: Based on Word Length (see Map).
   Example: "Fox" (Len 3) + Digit 5 → Fx5#
3. THE MAP: 1=! 2=@ 3=# 4=$ 5=% 6=^ 7=& 8=* 9=( 0=)

GRID RECOVERY (DIAGONAL SPREAD):
- Segment 1 is at Row [Digit 1], Column 0.
- Segment 2 is at Row [Digit 2], Column 1.
- Segment 3 is at Row [Digit 3], Column 2...
(The Column always matches the Sequence Count: 0, 1, 2...)
